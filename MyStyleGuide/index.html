Text Styles

H1: Main page heading or title

H1 heading style examples might be:

  • font-family: "Crimson Pro"
  • font-weight: bold
  • font-size: 50px

P: represents a paragraph element

Paragraph style examples might be:

  • font-family: "Pt Sans"
  • font-weight: 400 (regular)
  • font-size: 30px

Footer: contains info on the author, copyright data and links to other information

footer style examples might be:

  • font-family: "Pt Serif"
  • font-color: MachineGrey
  • font-size: 15px
  • anchor tags
